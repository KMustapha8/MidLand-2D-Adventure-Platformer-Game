# MidLand-2D-Adventure-Platformer-Game
A 2D adventure platformer built in Unity (C#) featuring player movement, collision mechanics, obstacle interaction, and a complete playable game loop exported as a Windows build.
MidLand is a standalone 2D platformer game developed using Unity and C#. The project includes core gameplay systems such as responsive player controls, scoring, collision detection, and level progression, demonstrating full-cycle game development from design to deployment.
